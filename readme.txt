Запуск приложения:
1) Получение данных через API: Terror-1.0.jar или Terror-1.0.jar -a
2) Получение данных из выбранного файла: Terror-1.0.jar -f "c:\\folder\\file.xml"
3) Получение данных из папки temp из самого свежего файла: Terror-1.0.jar -f


Пример: java -jar Terror-1.0.jar -f "c:\\folder\\file.xml"
